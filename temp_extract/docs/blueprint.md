# **App Name**: MahalluConnect

## Core Features:

- Admin Authentication: Secure login for admins via Firebase Authentication.
- Member Management: Add, edit, and bulk upload member data with block/cluster assignment.
- Transaction Recording: Record cash-in and cash-out transactions for members with remarks.
- SMS Notification: Automated SMS notifications via Fast2SMS (or MSG91) for each transaction using Firebase Functions as a tool, with monthly summary option.
- Universal Search & Passbook: Global search for members by various parameters; displays passbook with transaction history and summary.
- Reporting & Analysis: Generate financial reports with filters, summaries, and export options.
- Settings Management: Configure application settings like SMS API keys and manage admin accounts.

## Style Guidelines:

- Primary color: Deep green (#1B5E20) to convey trust and stability, in line with banking themes.
- Background color: Off-white (#F2F2F2) provides a clean, professional backdrop that ensures readability.
- Accent color: Soft blue (#5DADE2) used for interactive elements, providing clarity without overwhelming the design.
- Font: 'Inter' (sans-serif) for body text and headlines; provides a modern, neutral, and readable appearance.
- Flat vector icons with a finance theme for dashboard elements.
- Left sidebar navigation with a top header search bar for intuitive navigation.
- Subtle hover transitions for interactive elements.
