# JSB App - Mahallu Bank

A modern Next.js banking application built from scratch for transaction management and account operations.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Technology

- Next.js 15+
- TypeScript  
- Tailwind CSS
- Firebase
- React Hooks

## Build & Deploy

```bash
npm run build
npm start
```
